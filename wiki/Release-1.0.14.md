# Release 1.0.14

Version 1.0.14 erweitert die Navigations- und Admin-Funktionen des Themes und liefert ein sauberes Wartungs-Release.

## Highlights

- Theme-Version auf `1.0.14` angehoben
- Icon-Auswertung in der Navigation erweitert, damit Menue-Icons robuster aus Titel/Token abgeleitet werden
- Neue Admin-Seite unter Design fuer Icon-Browsing und Copy-Workflows hinzugefuegt
- README/Wiki auf den aktuellen Release-Stand synchronisiert
- Hotfix-Rebuild ohne Versionssprung: 1.0.14-ZIP ersetzt, damit Shared-Lib-Resolver fuer die Legal-Notice-Abhaengigkeit robust aufloest
- Theme-Release-ZIP enthaelt eine versionsgebundene Shared-Install-Payload fuer den zentralen Shared-Pfad

## Kompatibilitaet

- Theme-Version: `1.0.14`
- Keine beabsichtigten Breaking Changes
