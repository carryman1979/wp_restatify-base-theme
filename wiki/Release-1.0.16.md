# Release 1.0.16

Version 1.0.16 bringt die Shared-Aufloesung des Themes auf dieselbe Regelbasis wie die Plugins.

## Highlights

- Theme-Loader priorisiert lokal Root-Shared (`wp_restatify-shared/src/*`) fuer Development.
- Fallback nutzt nur die benoetigte Version unter `wp-content/plugins/wp_restatify-shared/versions/<x.y.z>/` (oder MU-Plugins).
- Theme-Version in `style.css` und `package.json` auf `1.0.16` angehoben.
- Copilot-Repo-Beschreibung auf Shared-Loader-Reihenfolge erweitert.

## Release-prep refresh (2026-05-30)

- Kein Versionssprung: Release-Prep verbleibt auf `1.0.16`.
- Footer-/Customizer-Aufraeumarbeiten und Theme-Look-and-Feel-Polish konsolidiert.
- Dark-Theme-nahe Feinanpassungen und Release-Build-Dokumentation fuer den Rollout nachgezogen.

## Kompatibilitaet

- Theme-Version: `1.0.16`
- Keine beabsichtigten Breaking Changes
