# Restatify Base Theme 1.0.12

## Was ist neu

- Neue Pattern-Kategorie `Restatify Blog` hinzugefuegt.
- Neue Core-basierte Blog-Starter-Patterns bereitgestellt:
  - Blog Hero
  - Blog Post Grid (Query Loop)
  - Related Posts
- Einheitliches Query-Loop-Card-System fuer den Restatify-Blog-Look integriert (Layout, Meta, Pagination, responsive Verhalten).
- Theme-/Readme-/Wiki-Beschreibungen auf den Standard-WordPress-Ansatz konkretisiert (Core-Blocks, Patterns, Templates, theme.json).

## Ziel

Ein sofort nutzbarer Blog-Workflow fuer Redakteure auf Basis von WordPress-Standardmechanismen ohne zusaetzliche Plugin-Abhaengigkeit.

## Kompatibilitaet

- Theme-Version: `1.0.12`
- Keine beabsichtigten Breaking Changes.
