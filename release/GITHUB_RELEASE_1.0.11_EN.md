# Restatify Base Theme 1.0.11

## What's new

- Switched header primary navigation to an assigned WordPress menu location.
- Added `Header Hauptmenue` menu location for explicit main navigation control.
- Added admin warning notice when no menu is assigned to the primary header location.
- Migrated primary navigation walker to `Walker_Nav_Menu` for robust menu/dropdown handling.
- Extended Contact block with flexible details list (text/link/button modes), item reordering, optional body text, and legacy field migration.
- Improved Contact block styles for link/button rendering and multiline content.
- Improved Oracle block typography with responsive word-size fitting and resize reflow.

## Included changes

- Header/navigation system update in theme setup, walker, and header pattern.
- Contact block source updates in block metadata, editor UI, and frontend/editor styles.
- Oracle runtime/style updates for stable one-line word rendering.

## Compatibility

- Theme version: `1.0.11`
- No intended breaking changes.
