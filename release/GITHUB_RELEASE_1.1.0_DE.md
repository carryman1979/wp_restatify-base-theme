# Restatify Base Theme 1.1.0

## Was ist neu

- Header-Hotfixs zu einem konsistenten Release gebuendelt.
- Header-Pattern um `navbar-actions` erweitert, damit Social/CTA/Theme/Language stabil ausgerichtet sind.
- Tagline-Verhalten korrigiert: Umbruch auf Desktop, kompaktere Typografie bei engen Breiten, Ausblendung im Mobile-Header.
- Kompaktmodus (992-1220px) ergaenzt: frueherer Wechsel in Hamburger/Collapse verhindert Ueberlagerung von Tagline und Menue.
- Opera-Desktop/Mobile-Hamburger robust gemacht (Pseudo-Element-Icon + Sichtbarkeits-/Layering-Fallbacks).
- Cookie-Consent + Booking-Fix: kein erzwungener Reload bei aktivem Booking-Overlay/hash-basierter Buchung.
- Neues Event `restatify:cookie-consent-changed` fuer Integrationen hinzugefuegt.
- Cookie-Banner-Z-Index oberhalb des Booking-Overlays angehoben.
- JS-Unit-Test fuer Consent-Booking-Guard und Test-Hooks hinzugefuegt.

## Kompatibilitaet

- Theme-Version: `1.1.0`
- Keine beabsichtigten Breaking Changes.
