# Restatify Base Theme 1.0.11

## Was ist neu

- Die Header-Hauptnavigation wurde auf ein zuweisbares WordPress-Menue umgestellt.
- Neue Menueposition `Header Hauptmenue` fuer eine explizite Steuerung der Hauptnavigation.
- Admin-Warnhinweis integriert, wenn fuer das Hauptmenue noch kein Menue zugewiesen ist.
- Primary-Navigation-Walker auf `Walker_Nav_Menu` migriert fuer saubere Menue-/Dropdown-Verarbeitung.
- Contact-Block erweitert um flexible Detailliste (Text/Link/Button), Sortierung der Eintraege, optionalen Beschreibungstext und Legacy-Migration alter Felder.
- Contact-Block-Styles fuer Link-/Button-Ausgabe und mehrzeilige Inhalte verbessert.
- Oracle-Block-Typografie verbessert mit responsivem Wort-Fitting und Resize-Reflow.

## Enthaltene Aenderungen

- Header-/Navigationssystem in Theme-Setup, Walker und Header-Pattern aktualisiert.
- Contact-Block-Quellcode in Block-Metadaten, Editor-UI sowie Frontend-/Editor-Styles erweitert.
- Oracle-Runtime und Styles fuer stabile einzeilige Wortdarstellung angepasst.

## Kompatibilitaet

- Theme-Version: `1.0.11`
- Keine beabsichtigten Breaking Changes.
