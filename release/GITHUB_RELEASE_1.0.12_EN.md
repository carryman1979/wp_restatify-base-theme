# Restatify Base Theme 1.0.12

## What's new

- Added a dedicated `Restatify Blog` block-pattern category.
- Added core-based blog starter patterns:
  - Blog Hero
  - Blog Post Grid (Query Loop)
  - Related Posts
- Added a global query-loop card styling system for consistent Restatify blog cards (layout, metadata, pagination, responsive behavior).
- Updated theme/readme/wiki descriptions to explicitly document the standard WordPress mechanism approach (core blocks, patterns, templates, theme.json).

## Goal

Provide a production-ready blog starter workflow that editors can use immediately with core WordPress tooling and no additional plugin dependency.

## Compatibility

- Theme version: `1.0.12`
- No intended breaking changes.
